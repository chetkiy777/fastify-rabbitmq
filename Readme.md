# Сервис очередей на Fastify + Typescript + RabbitMQ

### Сборка проекта, установка всех зависимотей, запуск проекта
```
make all
```

### Просто запустить контейнеры (если они уже собраны):
```
make start
```

### Посмотреть логи всех сервисов:
```
make logs
```

### Остановить сервисы:
```
make stop
```

### Полностью остановить и удалить контейнеры:
```
make down
```

### Полная очистка проекта (удаляет node_modules, dist и контейнеры):
```
make clean
```


## Для проверки сервиса отправить запрос:
> curl -X POST http://0.0.0.0:3000/events \
> ?-H "Content-Type: application/json" \
> -d '{"email":"test@mail.com"}'
